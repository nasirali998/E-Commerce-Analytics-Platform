# 🛒 E-Commerce Analytics Platform

## 📌 Project Overview

The **E-Commerce Analytics Platform** is a data analysis project built using **Python, NumPy, and Pandas**.
The main purpose of this project is to analyze e-commerce sales data, extract meaningful business insights, and understand customer, product, and revenue performance.

This project demonstrates real-world data analytics workflow including data loading, cleaning, merging, statistical analysis, and business intelligence.

---

## 🎯 Project Objectives

* Analyze customer purchasing behavior
* Calculate overall sales performance
* Identify top-performing products and customers
* Analyze payment methods
* Find revenue patterns by category and region
* Perform data transformation and feature engineering

---

## 🛠️ Technologies Used

* 🐍 Python
* 🔢 NumPy
* 🐼 Pandas
* 📄 CSV Dataset
* 💻 Jupyter Notebook / Python Environment

---

## 📂 Dataset Description

The project uses four different datasets:

### 1. Customers Dataset

Contains customer information:

* CustomerID
* Customer Name
* Region
* Customer Details

### 2. Orders Dataset

Contains order information:

* OrderID
* CustomerID
* ProductID
* Quantity
* Order Date

### 3. Products Dataset

Contains product details:

* ProductID
* Product Name
* Category
* Price

### 4. Payments Dataset

Contains payment information:

* PaymentID
* OrderID
* Payment Method
* Amount
